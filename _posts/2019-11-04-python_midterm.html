if 的用法¶

在程式中if是很重要的東西，因為你的程式要能夠因應不同的情況做不同事情

這時候我們就需要有個判斷式來決定說程式怎麼執行

那if的格式是這個樣子

if (條件):
    程式碼
elif(條件):
    程式碼
else:
    程式碼

也就是當條件成立(= True)時我們執行if之下的程式碼然後結束，條件不成立(=False)時判斷下一格內的條件

整個區塊只會有一個程式碼被執行到，另外elif跟else是選填

那重要的地方是if後面的程式碼需要縮排電腦才知道那些是在if裡面的程式

List Comprehension¶

這個東西是將利用迴圈將一個List做出來，這個東西對初學者比較難，多看幾個例子會比較好理解

看完上面的例子我們可以發現利用list comprehension可以讓我們的程式碼更加的簡潔

那它的使用方式是這樣

list = [(expression) (for loop) (if statment)]

expression只能出現一次，主要是決定list內的值的樣子，主要是將運算式寫在這裡,ex. i/2,i**3

而後面的for loop可以有多層 if判斷也可以有多個

以剛剛的例子來說

[i for i in range(3)]

我們先看後面的for loop(for i in range(3))，它會將i依序用0,1,2執行

而前面的表達式是i 所以就是後面的i填進去就可以了

所以整個list會是[0,1,2]

那我們接下來看幾個不同的例子
